# 🧬 BabyHippo DNA System

## ⚠️ 중요한 오해 해소

**이것은 "하드코딩"이 아닙니다.**

많은 사람들이 `threat_keywords`, `reflex_patterns`, `drive_weights` 같은 코드를 보고 "하드코딩이다", "규칙 기반이다", "진짜 뇌 모델이 아니다"라고 오해합니다.

**그러나 이것은 생물학적으로 정확한 설계입니다.**

## 🧠 생물학적 근거: 선천적 vs 후천적

### 실제 뇌에도 "하드코딩"이 존재합니다

| 유형 | 설명 | 예시 |
|------|------|------|
| **선천적 (Innate)** | 유전자에 새겨진 정보, 학습 없이 작동 | 뱀 공포, 높은 곳 공포, 빨기 반사 |
| **후천적 (Learned)** | 경험을 통해 시냅스가 변화하며 형성 | 언어, 얼굴 인식, 기술 |

### 신경과학 연구 근거

1. **선천적 공포 (Innate Fear)**
   - LeDoux (1996): 편도체의 일부 회로는 학습 없이 특정 자극에 반응
   - Ohman & Mineka (2001): 뱀, 거미에 대한 공포는 진화적으로 준비됨

2. **본능적 행동 (Instinctive Behavior)**
   - Tinbergen (1951): Fixed Action Patterns - 유전적으로 프로그래밍된 행동
   - 신생아의 빨기 반사, 파악 반사

3. **기질 (Temperament)**
   - Kagan (1994): 아기 때부터 나타나는 성격 경향은 유전적
   - 내향/외향, 불안 경향 등

## 📊 "하드코딩" vs "DNA 설정"의 차이

| 구분 | 하드코딩 (나쁜 예) | DNA 설정 (우리 코드) |
|------|-------------------|---------------------|
| **의도** | 개발 편의 | 생물학적 모델링 |
| **근거** | 없음 | 신경과학 논문 |
| **변경 가능성** | 코드 수정 필요 | 파라미터로 조절 가능 |
| **생물학적 대응** | 없음 | 선천적 반응 / 기질 |
| **예시** | `if name == "John": return "Hi John"` | `if threat in INNATE_FEARS: alert()` |

### config.py 구조
```python
# 1. 절대 법칙 (종 공통)
class FundamentalLaws:
    TABOOS = [...]           # 절대 금기
    SURVIVAL_PRIORITY = True # 생존 우선
    
# 2. 성격 유형 (개체 차이)

    QUOKKA = "quokka"
    SCHOLAR = "scholar"
    ...

# 3. DNA 템플릿 (세부 파라미터)
class DNA:
    def __init__(self, species):
        self.traits = {
            'drive_weights': {...},
            'emotional_bias': {...},
            'reflex_patterns': [...],
        }
```
