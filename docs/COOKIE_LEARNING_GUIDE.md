# 🍪 Cookiie 학습 가이드 - 뇌처럼 학습하기

## 📋 개요

Cookiie는 GPT·LLaMA 같은 대규모 언어 학습 방식이 **전혀 아닙니다**.

Cookiie는 **"뇌처럼"** 학습합니다.

즉, Cookiie의 학습은 다음 **4단계 구성**으로 이루어집니다:

---

## ✅ 1단계: 개념 뉴런 생성 (Concept Formation)

입력이 들어올 때 Cookiie는 먼저 **"개념 뉴런"**을 만듭니다.

### 예시

```
'사과' → 뉴런 N12
'빨간색' → 뉴런 N22
'달다' → 뉴런 N31
```

### 추천 입력 방식

```
학습: 사과
학습: 빨간색
학습: 달다
```

### 내부 동작

- 새 개념 뉴런 생성
- 선택성(selectivity) 조정
- 시냅스 가중치 초기화
- → Alpha Genome이 다듬어짐

---

## ✅ 2단계: 연결(Association) 형성 학습

뇌의 진짜 힘은 **"관계"**에서 나옵니다.

Cookiie도 마찬가지입니다.

### 예시

```
학습: 사과는 빨간색이다
학습: 사과는 달다
```

### 내부 동작

- "사과" 뉴런 ↔ "빨간색" 뉴런 연결 강화
- "사과" 뉴런 ↔ "달다" 뉴런 연결 강화
- Cookiie는 **STDP**로 연결을 직접 조절합니다

### 핵심 포인트

- ✔ 문장을 길게 쓰지 말 것
- ✔ 한 관계를 짧게 반복하는 것이 훨씬 효과적

---

## ✅ 3단계: 수면(Consolidation)으로 메모리 고정

학습을 많이 시킨 후엔 반드시 **수면**을 해줘야 합니다.

### 왜 필요한가?

- DG → CA3 → CA1 시냅스 재배선
- 중요도(weight) 재조정
- 단기기억 → 장기기억 전환(consolidation)

### 수면이 없으면?

- 기억이 흩어짐
- 불안정해짐
- 연결이 약해서 recall이 잘 안됨

즉, **"수면 = 모델 저장"**과 비슷한 개념입니다.

---

## ✅ 4단계: 질문으로 recall(회상) 테스트

학습 → 수면 후에는 질문해서 연결을 확인합니다.

### 예시

```
사과는 무슨 색이야?
사과는 어떤 맛이야?
```

### 내부 동작

Cookiie는 내부 기억을 검색해:

- 사과 → 빨간색
- 사과 → 달다

를 매칭합니다.

---

## 🎯 요약: Cookiie 학습 4단계

| 단계 | 목적 | 입력 예시 |
|------|------|-----------|
| 1) 개념 입력 | 개념 뉴런 생성 | 학습: 사과 |
| 2) 관계 입력 | 연합 생성 (STDP) | 학습: 사과는 빨간색 |
| 3) 수면 | 장기기억 고정 | 수면 |
| 4) 질문 | 회상 확인 | 사과는 어떤 맛? |

---

## 📌 반드시 지켜야 하는 규칙 (중요)

### ✔ 1) 한 번에 너무 많은 문장 주지 말 것

Cookiie는 길고 복잡한 문장을 모릅니다.

뇌처럼 **"짧은 자극"**을 많이 반복하는 게 훨씬 효과적입니다.

### ✔ 2) 같은 문장을 여러 번 주면 더 강화됨

STDP는 반복에 민감합니다.

**예시**:
```
학습: 사과는 빨간색
학습: 사과는 빨간색
학습: 사과는 빨간색
```
→ 아주 강한 연결 생성

### ✔ 3) 새로운 개념은 반드시 단독 입력 먼저

**예시**:
```
❌ 바로 "사과는 빨간색" 하지 마
⭕ 먼저 "학습: 사과" → "학습: 빨간색"
```

### ✔ 4) 수면은 필수

학습 5~10개 후 무조건 **"수면"**.

## 🚀 지금 바로 Cookiie에게 학습시켜볼 수 있는 예제

```python
from babyhippo.integration import CuriousBrain

cookiie = CuriousBrain(name="Cookiie")

# 1. 개념 뉴런 생성
cookiie.think("학습: 사과")
cookiie.think("학습: 빨간색")
cookiie.think("학습: 달다")

# 2. 연결 형성
cookiie.think("학습: 사과는 빨간색")
cookiie.think("학습: 사과는 빨간색")  # 반복으로 강화
cookiie.think("학습: 사과는 달다")

# 3. 수면
cookiie.sleep(cycles=5)

# 4. 회상 테스트
cookiie.think("사과는 어떤 색이야?")
cookiie.think("사과 맛이 뭐야?")
```

## 🔬 내부 메커니즘

### STDP (Spike-Timing Dependent Plasticity)

- 시냅스 가소성 메커니즘
- 반복에 민감
- 연결 강화/약화 자동 조절

### 해마 구조

- DG (치상회): 패턴 분리
- CA3: 패턴 완성
- CA1: 시공간 정렬
- SUB (서브쿨럼): 출력

### 수면 공고화

- 시냅스 재배선
- 중요도 재조정
- 단기 → 장기 전환
